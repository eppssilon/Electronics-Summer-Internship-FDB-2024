## Description

These projects were completed during the Electronics Summer Internship under FDB 2024.  
They include practical electronics designs (PCBs, schematics), reference materials, and finalized projects demonstrating the application of learned concepts in electronics engineering.

## How to Use

1. Explore the **Pictures/** folder to view circuit designs and schematics.  
2. Read the reports in **Report and presentation/** for detailed explanations.  
3. Refer to **Source Material/** for additional context and theory.  
4. Extract **The Final Projects/** ZIPs to access completed project files.
